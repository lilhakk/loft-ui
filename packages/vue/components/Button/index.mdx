# Button
Кнопка - элемент при нажатии на которую происходит какое-то действие

```js
import { Button } from 'loft-ui'
```

## Пример
```jsx
const [count, setCount] = useState(0);

return (
  <Button onClick={() => setCount(count + 1)}>
    Clicked {count} times!
  </Button>
)
```
