## Простой пример
```jsx
const [text, setText] = useState('');

return (
  <Input
    label='Email'
    value={text}
    onChange={v=> setText(text)}
  />
)
```

<Input
  style={{ width: 200 }}
  label='Email'
/>

## Задать маску
```jsx
const [text, setText] = useState('');

return (
  <Input
    label='Телефон'
    mask='+ 7 (999) 999-99-99'
    value={text}
    onChange={v=> setText(text)}
  />
)
```
